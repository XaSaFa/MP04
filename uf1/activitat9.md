# Activitat 9 - Consultora de serveis informàtics.

Per aquesta activitat buscaràs una manera de crear una pàgina web online de forma gratuïta (pots fer servir google sites, wordpress.com, wix...).

Aquesta pàgina tindrà una portada amb un títol dient "Consultora X" on X és el teu cognom i la frase "Serveis informàtics".

També tindrà una imatge relacionada amb la temàtica (serveis de consultoria informàtica).

Hi haurà una secció BEN VISIBLE que indicarà "Formulari d'incidències informàtiques", que serà un link a un formulari FUNCIONAL per als usuaris de la consultora.

Aquest formulari servirà per què els usuaris/es amb problemes es possin en contacte amb el servei tècnic.

El formulari preguntarà:

- La direcció de correu electrònic.
- El nom complert de l'usuari/a.
- Quin SO té el client (podrà escollir entre Windows o Linux).
  - Si un usuari escull Windows anirà a una secció on se li deixa escollir entre:
    - Windows 7.
    - Windows 8.
    - Windows 10.
    - Windows 11.
  - Si escull Linux anirà a una secció on se li deixa escollir entre:
    - Linux Mint 21.2
    - Ubuntu Desktop 22.04 LTS
    - Ubuntu Desktop 23.10
    - Altres.
  - Si escull altres se li deixa un camp per indicar quina distribució de Linux té.
- Per últim se li pregunta quin és el seu problema i se li deixa un camp de text gran per respondre i poder enviar el formulari.   

S'ha d'entregar el link a la web amb el formulari operatiu.
